Netflix-Clone-HTML-CSS
A Netflix UI clone built using HTML and CSS. This project replicates the Netflix homepage design for learning front-end web development.


 Features
 
- Netflix-like homepage UI
- Responsive design (works on mobile & desktop)
- Background images and videos
- Hover effects on buttons and cards
- Clean and modern layout


 Technologies Used
 
- HTML5  
- CSS3  
- No JavaScript (Pure HTML & CSS)


 Learning Purpose
 
This project is created for educational purposes only.
All design inspiration is taken from Netflix, but this is not an official Netflix website.


 How to Run This Project
 
Open the folder in VS Code or any code editor.
Open index.html in your browser.
